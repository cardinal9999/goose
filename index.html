Goose

Goose is an online tool for compression. You can compress different types of data.

How it Works

Goose works by replacing common substrings found in nodes with ASCII control characters. Suppose we have this JSON node: {"userid": 4545, "icons": ["/img/4545_16.png", "/img/4545_32.png"], "status": null, "deleted": false} It is hard to believe the object is 101 characters long. We can compress this by replacing some common substrings with ASCII characters. This string is only 88 characters long. If you want to decompress, you can use the decompression tool.

Collisions

A collision occurs when the decompressor detects an ASCII character that is not a compressed substring.This can cause problems in decompression because the decompressor parses ASCII characters as keywords. Escape characters are the solution. For node types that do not have escape characters, you can encode them with Base64.
